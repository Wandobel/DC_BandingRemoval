# DC_BandingRemoval

Script pour PixInsight permettant de supprimer les bandes verticales et horizontales
(banding) sur les images astronomiques.

## Installation
Copier le fichier **DC_BandingRemoval.js** dans :
`PixInsight/src/scripts`

Puis relancer PixInsight.

## Utilisation
Menu : **Script → Utilities → DC Banding Removal**  
Permet de traiter les bandes verticales ou horizontales avec un facteur ajustable.
