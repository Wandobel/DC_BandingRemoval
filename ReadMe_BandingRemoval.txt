# DC_BandingRemoval


### Description

DC_BandingRemoval est un script PixInsight permettant de supprimer les bandes  
verticales et horizontales (banding / vertical shift) visibles sur certaines  
images astronomiques, sans dégrader les étoiles ni les structures fines.

DC_BandingRemoval is a PixInsight script designed to remove vertical and horizontal banding artifacts 
(banding / vertical shift) visible in certain astronomical images, 
without degrading stars or fine structural details.
